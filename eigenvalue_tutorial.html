如何求解矩阵的特征值？

以矩阵 A = [[2, 2], [1, 3]] 为例

核心思想：从 A&vec;v = λ&vec;v 出发

我们通过一系列代数变形，将求解特征向量 &vec;v 和特征值 λ 的问题，转化成一个我们熟悉的问题。

A&vec;v - λI&vec;v = 0 => (A - λI)&vec;v = 0

关键突破口：为了让向量 &vec;v 有非零解，矩阵 (A - λI) 的行列式必须为 0。

因此，我们的目标就是解这个**特征方程**： det(A - λI) = 0

第二步: 计算行列式 det(A - λI)

我们计算上一步得到的新矩阵的行列式，使用 ad - bc 法则。

det(A - λI) = (2-λ)(3-λ) - (2)(1)

展开多项式：

= (6 - 2λ - 3λ + λ²) - 2

化简得到一个关于 λ 的二次多项式，我们称之为“特征多项式”：

= λ² - 5λ + 4

第三步: 求解特征方程

我们令上一步得到的特征多项式等于0。

λ² - 5λ + 4 = 0

通过因式分解：

(λ - 1)(λ - 4) = 0

解出特征值: λ₁ = 1, λ₂ = 4

结论

我们已经成功解出了矩阵A的两个特征值：1 和 4。

这告诉我们，矩阵A这个变换，在空间中有两个“特殊方向”（特征向量所在的方向）。在一个方向上，它不进行任何缩放（因为λ=1）；在另一个方向上，它将向量的长度拉伸为原来的4倍（因为λ=4）。
